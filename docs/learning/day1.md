## 遇到的问题与解决方案
1. **问题**: Git安装后命令不可用
   **解决**: 重启终端或添加Git到系统PATH

2. **问题**: Docker Desktop启动失败
   **解决**: 检查Hyper-V或WSL2是否启用

3. **问题**: Python版本不符合要求
   **解决**: 从官网下载指定版本安装
